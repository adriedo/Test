Branch Level Classifier

Monthly delinquency-based branch classification logic

900 · Downgrade 901 · Baseline 902 · Upgrade
Downgrade — entry
A branch enters downgrade when…
A Misses Standard — actual delinquency exceeds allowable balance
— or —
B Hits Standard but misses both NB and PB-Small in the same month
Both must be missed simultaneously. If Total is missed, condition A fires instead.
Downgrade — exit
A branch exits downgrade when…
1

Hits Standard for 3 consecutive months
Running streak resets to 0 on any miss

or
2

Both NB and PB-Small hit standard for 3 consecutive months
Each streak is tracked independently and both must reach 3

Exit check fires first — if exit criteria are met in the same month a goal is missed, the branch exits downgrade.
⚑
Account count gate — A branch must retain ≥95% of its prior-year account count to be eligible for downgrade status. Branches that have significantly shrunk are excluded regardless of delinquency performance.
Upgrade eligibility
A branch qualifies for Level 902 when not downgraded and all conditions below are met
Tier Grades Min booking rate Min FPS rate Window
A/B A + B ≥ 70% ≥ 90% Rolling 12 months

All 3 tiers must be
met simultaneously
B/C B + C ≥ 65% ≥ 80%
C/FS C + FS ≥ 60% ≥ 75%
Additionally requires 3 consecutive months hitting Standard
◆ IL and NM regions use a regional FPS benchmark (regional average + 1%) instead of fixed thresholds above
◆ Online applications are discounted 30% when calculating the adjusted booking rate
